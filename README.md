# Allergenic properties of Berlin grasslands

*Author*: Maud Bernard-Verdier
*Collaborators*: Birgit Seitz, Sascha Buchholz, Ingo Kowarik & Jonathan Jeschke

This repository contains the code and data to reproduce the analyses in the manuscript entitled "Allergens in grasslands increase with urbanisation and plant invasions", currently in preparation.This code analyses the allergenic properties of 56 plots of dry acidic grasslands in Berlin, Germany. This research work is part of the BIBS project, **Bridging in Biodiversity Science**, funded by the BMBF, Germany.
